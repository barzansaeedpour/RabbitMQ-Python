# RabbitMQ-Python

## Install RabbitMQ

- install Erlang
    - https://www.erlang.org/downloads

- install RabbitMQ server
    - https://www.rabbitmq.com/docs/install-windows#downloads

- list of RabbitMQ plugins
    - run RabbitMQ command prompt 
    - ```
        rabbitmq-plugins.bat list
        ```
